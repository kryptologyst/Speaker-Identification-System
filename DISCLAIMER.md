This software is provided for:

- Understanding audio processing and machine learning

## Prohibited Uses

**DO NOT use this system for:**
- Production biometric identification
- Unauthorized voice cloning
- Surveillance without consent
- Any commercial biometric applications
- Violation of privacy rights
- Identity theft or fraud

## Privacy Considerations

### Data Handling
- Audio data is processed locally
- No data is transmitted to external servers
- Filenames and metadata are anonymized in logs
- No personally identifiable information is stored

### Limitations
- This is a demonstration system
- Accuracy may not be suitable for production use
- No guarantees about security or privacy in production environments
- Requires proper consent and legal compliance for any real-world use
